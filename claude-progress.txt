# Fintax AI - 项目进度

## 已完成的功能

### 1. project_bootstrap (2026-02-26)
- 创建后端工程：Java 21 + Spring Boot 3.2.5 + MyBatis-Flex + PostgreSQL + Flyway + Lombok + Fastjson2
- 创建前端工程：Vue 3 + TypeScript + Vite + Tailwind CSS + Vue Router
- 实现左侧菜单布局：Accounts -> Account, Crypto -> Transactions
- Apple Dark 风格基础 UI（暗色侧边栏、内容区域）
- 后端健康检查接口 GET /api/health
- CORS 配置允许前端 localhost:5173 访问
- 统一 API 响应格式 Result<T> (code, message, data)
- init.sh 启动脚本

### 2. db_schema (2026-02-26)
- Flyway V2~V5 迁移脚本：main_account, sub_account, crypto_tx, asset_reconcile_snapshot
- 所有表含 is_deleted 软删、created_at/updated_at 时间戳
- crypto_tx 去重唯一索引 uk_tx_dedupe（9 列）
- asset_reconcile_snapshot 唯一约束（sub_account_id + currency + snapshot_time）

### 3. backend_models (2026-02-26)
- 4 张表的 Entity（Lombok）+ Mapper（MyBatis-Flex BaseMapper）
- MainAccountService / SubAccountService：CRUD + 软删 + 分页
- CryptoTxService：多条件分页、按 tx_hash 查明细、批量插入、聚合分页、批量更新
- AssetReconcileSnapshotService：保存/批量保存、分页查询、导出查询

### 4. account_api (2026-02-26)
- MainAccountController：POST/PUT/DELETE/GET/分页 + Excel 导入接口
- SubAccountController：POST/PUT/DELETE/GET/按主账户查询/分页
- 删除校验：删除主账户前检查子账户和交易记录
- 完整 Req/Resp DTO 和 JavaDoc

### 5. account_import_excel (2026-02-26)
- ExcelImportService：解析 .xlsx、自动创建 Sub Account、批量写入 crypto_tx
- 表头别名匹配（支持多种列名格式）
- UUID import_batch_id 生成
- 逐条插入处理 uk_tx_dedupe 去重
- ImportResultResp 返回成功/失败条数和失败详情

### 6. transactions_api (2026-02-26)
- CryptoTxController：Tx Group 分页聚合（GROUP BY tx_hash）+ Tx Details 明细
- 聚合输出：Category（Send/Receive/Mixed）、From/To、Account Name、Inflow/Outflow/Fee
- Description 和 Business Type 按 tx_hash 批量编辑接口

### 7. assets_reconcile_api (2026-02-26)
- AssetReconcileController：生成快照 + 列表查询 + Excel 导出
- system_total = Σ(Receive) - Σ(Send)，onchain_balance 使用 mock 值
- diff/diff_abs/diff_status 自动计算

### 8. ui_design_system (2026-02-26)
- Tailwind 扩展 Apple Dark 主题色（apple-bg/separator/accent/green/red 等）
- 全局样式：自定义 scrollbar、apple-table 组件类
- 组件库：AppButton(4 variant) / AppInput / AppSelect / AppModal / AppToast / AppDropdown / AppTag(6 color)
- 系统字体 SF Pro + 抗锯齿渲染

### 9. ui_account_page (2026-02-26)

## 后端 API 列表
- GET /api/health - 健康检查
- POST /api/main-accounts - 创建主账户
- PUT /api/main-accounts/{id} - 编辑主账户
- DELETE /api/main-accounts/{id} - 删除主账户（软删）
- GET /api/main-accounts/{id} - 查询主账户
- GET /api/main-accounts - 分页列表
- POST /api/main-accounts/{id}/import - Excel 导入交易
- POST /api/sub-accounts - 创建子账户
- PUT /api/sub-accounts/{id} - 编辑子账户
- DELETE /api/sub-accounts/{id} - 删除子账户
- GET /api/sub-accounts/{id} - 查询子账户
- GET /api/sub-accounts/by-main-account/{mainAccountId} - 按主账户查子账户
- GET /api/sub-accounts - 分页列表
- GET /api/crypto-tx/groups - Tx Group 聚合分页
- GET /api/crypto-tx/details - Tx Details 明细
- PUT /api/crypto-tx/description - 编辑 Description
- PUT /api/crypto-tx/business-type - 编辑 Business Type
- POST /api/reconcile/generate - 生成对账快照
- GET /api/reconcile/list - 对账列表
- GET /api/reconcile/export - 导出对账 Excel
